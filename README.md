## Installation

LWDrawboard is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'LWDrawboard'
```

**Carthage**
```ruby
github "luowei/LWDrawboard"
```
